# MoneyIQ

## Screens:

1. Home Screen - Shows live stock news in a 3 column layout with stock prices of top 10 stocks somewhere in between
2. Login Screen - accepts username and password
3. Signup Screen - accepts name, username, email id and password
4. KYC Screen - accepts aadhaar card number, phone number
5. User Dashboard - shows live stock, ipo and mutual fund prices in a floating navbar with 3 tabs
6. Admin Dashboard - has a floating navbar with 2 tabs, the first tab allows to upload new expert videos (from either youtube or from device), the second tab allows to add todays expert advice(s) and also shows history of all the previous advices
7. User Profile - has a sidebar with 2 tabs, the first tab has the user details - name, username, password, email id, phone number - and has options to update each one too (after some kind of verification via email or phone number). the second tab has the user's portfolio in a pie chart layout showing the composition of each bought stock. There are some metrics below the pie chart as well
